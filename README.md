Pac-Kart is an in progress file editor.

6 Games are known to use this format.
Motor Mayhem: Vehicular Combat League
Hot Wheels Velocity X
Snoopy vs. the Red Baron
Pac Man World Rally
Bee Movie Game
Bigfoot: Collision Course

The goal is to have each game and version extractable and fully editable.
Currently only Pac Man World Rally PC is fully extractable and partially editable.

The Pac Man World Rally Save file is also editable.
